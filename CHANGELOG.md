# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-01

### Added

- Initial release
- Start/stop recording commands with keyboard shortcuts
- Automatic file change detection and diff generation
- Manual AI prompt/response capture via multiline input
- Note-taking functionality with optional tags
- Status bar UI with recording timer
- `.buildlog` file export in JSON format
- Initial and final file snapshots
- Configurable output directory
- API key configuration for future upload feature
